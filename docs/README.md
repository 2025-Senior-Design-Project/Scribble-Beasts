# Project Documentation

Welcome to the Scribble Beasts documentation. This folder contains user-facing guides and a short FAQ to help new players and operators get started.

Available pages

- [User Guide](USER_GUIDE.md) — quick start, how to play, and troubleshooting for end users.
- [User Manual](USER_MANUAL.md) — deeper reference for game controls, configuration, and developer-facing tips.
- [FAQ](FAQ.md) — short answers to common questions.

Screenshots and images

Place any screenshots or illustrative images in the `docs/images/` folder and reference them from the pages above using relative paths (for example `images/example.png`).

If you'd like this site to appear on your project website (for example GitHub Pages), you can point the pages here to the published docs directory.
